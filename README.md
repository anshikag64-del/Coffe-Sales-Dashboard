# Coffee Sales Dashboard

## Overview
An interactive Excel dashboard created to analyze coffee sales performance using Pivot Tables, Pivot Charts and Slicers.

## Features
- Sales Trend Analysis
- Top Customers
- Country-wise Sales Analysis
- Coffee Type Comparison
- Interactive Filters

## Tools Used
- Microsoft Excel
- Pivot Tables

- Slicers
- Dashboard Design
